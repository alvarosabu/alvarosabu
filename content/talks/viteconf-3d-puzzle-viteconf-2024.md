# How the ViteConf 3D Puzzle was Built

A behind-the-scenes look at building the interactive 3D puzzle for the ViteConf landing page using TresJS and Vue, showcasing the collaboration between StackBlitz and TresJS.

This talk explores the journey of creating the 3D puzzle featured on the ViteConf 2024 landing page. The interactive puzzle, which users can solve via an integrated code editor, was built using TresJS, demonstrating the power of combining Vue, Vite, and 3D graphics for creating engaging web experiences.

Learn about the technical challenges, design decisions, and implementation details that went into creating this unique interactive experience for the Vite community.
