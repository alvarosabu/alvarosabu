# What's Hot on TresJS v4

A full update on what we've been developing for v4 of the core, including new devtools, translated docs, a cookbook, new features and extensive bug fixing.

Version 4 introduces performance improvements, on-demand rendering, typing support, and memory management. Event bubbling and primitive creation are key features, along with enhanced memory management and scene inspection capabilities.

TresJS has gained popularity with over 1,700 GitHub stars, 9K monthly NPM downloads, and a strong developer community.
