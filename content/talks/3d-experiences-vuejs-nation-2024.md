# How to Create 3D Experiences Using Vue with TresJS

Discover how the Tres ecosystem reduces the gap between Vue developers and the world of WebGL while enhancing the developer experience of working with 3D on the browser.

This talk explores how TresJS, a custom render API, enables developers to build 3D scenes in Vue using components and composables. During the presentation, a small 3D scene was built live to demonstrate the capabilities of TresJS.

TresJS provides an excellent introduction to creating 3D experiences using Vue, making complex 3D development accessible to Vue developers.
