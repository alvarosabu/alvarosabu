# TresJS, a declarative way of creating 3D scenes from Vue components

Meet TresJS ▲ ■ ●, a declarative way of bringing the magic of ThreeJS using everyday Vue Components and composables.

In this talk, I introduce TresJS as a library for creating 3D scenes using Vue components, highlighting how it's a lightweight and TypeScript-supported solution for developers interested in 3D development without dealing with the complexities of WebGL.

The presentation covers the process of setting up a 3D scene and objects in TresJS, along with creating and configuring 3D objects declaratively.
