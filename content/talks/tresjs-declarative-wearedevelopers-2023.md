# TresJS: A New Declarative ThreeJS as Vue Components

TresJS offers a declarative, always up-to-date way to use Three.js in your app. Build complex Three.js scenes declaratively using standard Vue components and composables.

This presentation covers how TresJS allows you to build complex Three.js scenes declaratively using standard Vue components and composables, and how TresJS automatically generates components from Three.js constructors.
